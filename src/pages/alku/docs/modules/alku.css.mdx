# alku.css module

This page is a bit different from the other pages on this site. This is what things look like when you don't import in
the other parts of Alku and instead opt for only the part that &ndash; for the most part &ndash; gives similar features
to what `normalize.css`, `sanitize.css`, and `modern-normalize.css` provide.
